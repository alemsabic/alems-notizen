# Atomarität als Kernprinzip des Zettelkastens

---

## 1. Inhalt (Der Gedanke)

Jeder Zettel enthält nur eine einzige, abgeschlossene Idee. Das ist das Prinzip der Atomarität.

Das Prinzip löst ein fundamentales Problem: Wenn eine Notiz mehrere Ideen enthält, kann sie nur an einem Ort im System existieren. Verknüpfungen werden mehrdeutig. Welcher Teil der Notiz ist gemeint, wenn man darauf verweist?

Atomare Notizen sind kombinierbar. Drei separate Zettel können in beliebiger Reihenfolge verknüpft werden. Eine Notiz mit drei Ideen ist fixiert. Sie hat genau eine Struktur, die nicht aufgebrochen werden kann, ohne die Notiz zu zerstören.

Atomarität schafft Klarheit. Der Titel beschreibt genau, was drin steht. Keine Überraschungen, keine Ablenkungen. Ein Gedanke, ein Zettel.

Die Praxis zeigt: Notizen unter 300 Wörtern sind meist atomar. Über 1000 Wörter enthält die Notiz fast immer mehrere Ideen, die getrennt werden sollten.

---

## 2. Herkunft (Quelle)

*   **Quelle:** Niklas Luhmann, Zettelkasten-Methode (Kernprinzipien)
*   **Original-Kontext:** Das Prinzip der Atomarität wird als erstes der Kernprinzipien genannt. Es ist die Grundlage für Kombinierbarkeit und Klarheit im System. Ohne Atomarität funktioniert die Vernetzung nicht, weil unklar ist, worauf Verweise sich beziehen.
